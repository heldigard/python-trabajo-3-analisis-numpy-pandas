# Trabajo 3: Análisis de datos con NumPy y Pandas

Objetivo:
Realizar análisis descriptivo de un dataset usando NumPy y Pandas.

Requisitos:
- Cargar un CSV de ejemplo (`data/`) y realizar limpieza básica.
- Calcular estadísticas descriptivas (media, mediana, desviación).
- Agrupaciones (`groupby`), filtrado y creación de nuevas columnas.
- Entrega en un notebook `analisis.ipynb` o en un script `analisis.py` más `reports/`.

Entrega:
- `data/` con dataset de ejemplo, `analisis.ipynb` o `analisis.py`, `README.md`.
